When you set lag = 336, it predicts the data better. In the other graph, it shows what happens when you set lag = 300. Since we are using as predictors their lagged values, the longer the lag, the more data is being used to model the data. In the scenario with lag=300, we can see that the model trails the forecast as there is less of a lag used.
